## Mod Overview
This mod makes changes to Beetle starting stats with the intent to make them somewhat more of a formidable opponent as opposed to a complete, pitiful joke.

### Changes
Beetle stats:
Base Attack Speed: 1 => 2
Base Movement Speed: 6 => 10
